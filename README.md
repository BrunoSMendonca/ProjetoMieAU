# Projeto Mi&Au - Website Institucional

Este é o site é uma proposta de site oficial para  **ONG Projeto Mi&Au**, focado em promover a conscientização sobre a castração e facilitar o contato para adoção e doações.

O projeto nasceu da necessidade de criar uma presença digital sólida para a instituição, apresentando uma interface amigável e funcional para os amantes de animais.

# Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as tecnologias fundamentais da web:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização, layout responsivo e efeitos visuais (Flexbox, Drop-shadow, Gradients).
- **Font Awesome**: Ícones para integração com redes sociais.
- **Google Fonts**: Tipografia focada em legibilidade.

# Funcionalidades Implementadas

- **Topbar**: Acesso rápido ao WhatsApp e Instagram da ONG.
- **Header Principal**: Menu de navegação intuitivo com efeito hover.
- **Seção Hero**: Destaque para a campanha de castração com tratamento de imagem via CSS.
- **Design Responsivo**: Planejado para se adaptar a diferentes tamanhos de tela.

# Identidade Visual

As cores foram escolhidas para refletir os valores da ONG:
- **Laranja (#F39200)**: Energia, calor e urgência para chamadas de ação (botões).
- **Azul (#E0F4FF)**: Confiança, tranquilidade e cuidado com os animais.
